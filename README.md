# Nevin Raju — Portfolio

Personal portfolio website showcasing my experience, projects, and skills as a full-stack developer.

## Tech Stack

| Layer | Technology | Version |
|-------|-----------|---------|
| Framework | [React](https://react.dev) | 19 |
| Language | [TypeScript](https://www.typescriptlang.org) | 5.8 |
| Build Tool | [Vite](https://vite.dev) | 6 |
| Styling | [Tailwind CSS](https://tailwindcss.com) | 4 |
| Text Layout | [@chenglou/pretext](https://github.com/chenglou/pretext) | 0.0.5 |
| Icons | [Lucide React](https://lucide.dev) | latest |

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | TypeScript check + production build |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview production build |
